Overview

Pivotal Sprint Tracker provides greater visibility into Sprints that are managed using Pivotal Tracker

Conventions:

  • Each sprint is grouped by a single label/epic so that it can pulled in a search
  • Done is a special custom field. It's value is set to 0 or 100 depending on the current state of the story. To set a value for done add ^done=40% say, on a new line in the description of the story
